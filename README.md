## 🚀 Deployment

האתר מתעדכן **אוטומטית** בכל push ל-`main`:

```bash
git add .
git commit -m "עדכון תוכן"
git push origin main
```

GitHub Pages בונה ומעלה תוך **1-2 דקות**.
